# OpenPort 🔍

A lightweight, multi-threaded Python port scanner with banner grabbing and file export.

> *"No bullshit. Just open ports."*

---

## 🚀 Features

- Multi-threaded scanning (fast)
- Supports domain and IP input
- Banner grabbing from open ports (HTTP, SSH, etc.)
- User-friendly defaults (just press Enter)
- Saves results to timestamped `.txt` files
- No external dependencies — just Python 3
